# Introduction
Sample project to invoke [reactive-audit](https://github.com/octo-online/reactive-audit) directly with

* Maven
* Gradle

Builder| Run | Audit
:-- | ---| ---
Maven | $ mvn exec:java -Prun | mvn audit.log -Paudit
Gradle | $ gradle run | $ gradle audit
SBT | $ sbt run | $ sbt audit
